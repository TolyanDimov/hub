# 3D Canvas-версия

Эта версия не использует Plotly, WebGL или внешние библиотеки.

## Запуск

1. Откройте `riemann_zodiac_3d_canvas.html` двойным кликом.
2. Вращайте сцену левой кнопкой мыши.
3. Масштабируйте колёсиком.
4. Включайте и выключайте слои слева.

## Повторная генерация

```bash
python generate_riemann_zodiac_3d_canvas.py
```

Нужен только Python 3. Дополнительные пакеты не требуются.

## Почему эта версия надёжнее

Прошлая версия использовала Plotly и WebGL. Некоторые браузеры или настройки
аппаратного ускорения могут не создавать WebGL-контекст для локального HTML.
Здесь 3D-проекция рассчитывается вручную и рисуется обычным Canvas 2D.
